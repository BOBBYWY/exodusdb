System Configuration File

Scope The scope of configuration can be per database, per installation (recommended), or per server.
A EXODUS installation sometimes has multiple databases, and in a few cases a server may have multiple EXODUS installations.
Any item which is configured in more than one scope gets the configuration of the HIGHER (broader) scope only
so any item configured at a lower (narrower) scope will be IGNORED if the item is also configured at a higher scope.
