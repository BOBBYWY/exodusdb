EXODUS Keyboard Shortcuts

General

Ctrl+Enter or F9- SAVE/YES/START - Commit any screen or popup
Esc or F8- CANCEL/NO/STOP - Close any document or popup
F2 - EDIT - Edit the current field instead of replacing it
Alt+M - MENU - Open the menu. Press letters or cursor keys to select menu items
F7 - POPUP - Options for the current item
F6 - LINK - Open related screen for current item

Documents

Alt+N - NEW - Start a new document
Alt+O - OPEN/SEARCH - Open an existing document - Search/Browse/List
Alt+C - CLOSE - Close the current document
Alt+E - EDIT/RELEASE - Edit document/Release document to other users
Alt+P - PRINT - Print/Send the current document.
Alt+L - LIST - Get a list of the documents.
Alt+K - KEY - Move to the first field on the document
Alt+0 Alt+1 etc. - BUTTON0-9 - Special buttons per form

Rows and Columns

Ctrl+Shift+N - NEW LINE - Insert a new line (Ctrl+N does not work)
Ctrl+D - DELETE LINE - Delete the current line (or Ctrl+Shift+D)
Alt+Y - COPY DOWN - Duplicate the data from the same column in the row above
Ctrl+G - GO TO - Go to line number xx/Show current line number
Ctrl+Home - FIRST COLUMN - Go to the first column in a table
Ctrl+End - LAST COLUMN - Go to the last column in a table
